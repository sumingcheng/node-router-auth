# node-router-auth
node 实现完整的 JWT 登录流程
# 项目构建流程
1. 安装 
```
npm i express mongoose typescipt bcrypt jsonwebtoken dotenv
```
2. 安装类型注解，因为要使用到TS 
```
npm i ts-node-dev @types/express @types/mongoose @types/node @types/jsonwebtoken @types/bcrypt -D
```
3. 初始化TS配置文件
```
npm install -g typescript
tsc --init
```
